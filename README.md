# Installation

1. Clone Moviideck

          git clone https://github.com/Ak-ram/Moviideck.git

2. Navigate to Moviideck directory

          cd ./Moviideck
          
3. Install dependences

          npm i
          
4. Run Moviideck

          npm run start
